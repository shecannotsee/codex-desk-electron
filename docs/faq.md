# FAQ

## 1. 为什么提示“未找到 codex 命令”？

原因：系统 PATH 中没有可执行 `codex`。

处理：

1. 终端执行 `codex --version`。
2. 检查 `Codex命令` 配置是否可执行。

## 2. 为什么发送时报“目录不存在”？

原因：当前 `工作目录` 不存在或不可访问。

处理：

1. 切换到存在目录。
2. 确认目录权限。

## 3. 为什么模型显示未知或获取失败？

模型探测为多路合并（命令参数 / 配置文件 / 运行 JSON / 主动 probe）。

处理：

1. 命令中显式加 `--model`。
2. 在“设置 -> 运行 -> 获取模型”刷新。

## 4. 运行中关闭窗口会怎样？

会弹确认框：

1. 取消
2. 停止任务并关闭
3. 直接关闭

建议优先选“停止任务并关闭”。

## 5. 为什么页面上少了以前的一些按钮？

功能被整理到两个入口：

1. 左侧会话右键菜单（新建/重命名/关闭）。
2. 右上角“设置”多级菜单（对话/运行/视图/编辑/应用/帮助）。

## 6. “视图缩放”和“对话字号”是比例联动吗？

不是。两者独立，最终视觉效果叠加：

- 视图缩放：窗口级
- 对话字号：内容级

## 7. 运行步骤为什么默认折叠？

为了在长任务中保持可读性。可逐条展开查看。

## 8. 排队的消息在哪看？

切到右侧“运行步骤”标签，顶部有“待执行排队消息”面板，显示内容、顺序和入队时间。

## 9. 怎么切换浅色/深色？

“设置 -> 视图 -> 主题”。

## 10. 左侧会话能调宽度吗？

可以。拖动左侧会话区与主区之间的分隔线即可，宽度会持久化保存。

## 11. 输入框字多了会自动换行吗？

会。输入框使用多行 `textarea`，会按边界自动换行。

## 12. 我在 Windows/macOS 上可以直接用吗？

当前状态：

- 已验证：`Ubuntu 22.04`
- 未验证：`Windows`、`macOS`

## 13. 数据存在哪里？

- 会话状态：`.codexdesk/state.electron.json`
- 兼容旧目录：`~/.codexdesk/state.electron.json`
- UI 偏好：浏览器本地存储 `codexdesk.ui-prefs.v1`

## 14. Ubuntu 上怎么安装成桌面应用？

使用 `.deb`：

1. 在 `src` 执行 `npm run dist:deb`
2. `sudo apt install ./dist/Codex\ Desk-*.deb`
3. 应用菜单搜索 `Codex Desk` 或执行 `codex-desk`

详见：`docs/deploy-ubuntu.md`。

详见：`docs/uninstall.md`。
